Array.isArray(value)-- 判断 value 是否为数组

遍历：

arrayObject.forEach(callback)

　　　-- 对数组的每个元素执行一次提供的函数

　　　-- callback 是一个函数

　　　-- callback(currentValue, index, array)

　　　　　currentValue : 数组中遍历到的当前值

　　　　　index : 当前值在数组中的下标

　　　　　array : 调用 forEach() 方法的数组

　　　　　注意: 没有返回一个新数组! 没有返回值!

arrayObject.map(callback)

　　　-- 对数组的每个元素执行一次提供的函数后返回其结果。

　　　-- 返回一个新数组，由所有返回的结果组成。

            var newarr = arr1.map(callback)

arrayObject.some(callback)

　　　-- 测试数组中的某些元素是否通过由提供的函数实现的测试。

　　　示例：

            [2, 5, 8, 1, 4].some(function(curr){
                                return curr 9;
            })　// false

            [2, 5, 8, 1, 4].some(function(curr){
                                return curr 7;
            })　// true

arrayObject.every(callback)

　　　-- 测试数组的所有元素是否都通过了指定函数的测试。和some()一样用法

arrayObject.filter(callback)

　　　-- 返回一个新数组, 其包含通过所提供函数实现的测试的所有元素。

　　　示例：

            [2, 5, 8, 1, 4].filter(function(curr){
                            return curr > 4;
            })　// [5,8]

查找：

indexOf(searchElement[, fromIndex])

　　　-- 查找数组中某元素第一次出现的索引

　　　-- 返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1。

　　　-- searchElement: 待查找元素

　　　-- fromIndex: 起始查找索引

　　　示例：

            [2, 5, 8, 1, 4].indexOf(8)  // 2
            [2, 5, 8, 1, 4].indexOf(8,3) // -1,因为从索引 3 开始，找不到8

lastIndexOf()

　　　-- 查找数组中某元素最后一次出现的索引,不传参数返回-1

            [2,5,4,6,5,8,7,6,5].lastIndexOf(5) // 8

归并：

reduce(callback,initialValue)

　　　-- 对累加器和数组中的每个元素（从左到右）应用一个函数，将其减少为单个值。

　　　-- callback(accumulator,curr,currIndex,array)

　　　-- accumulator 累加器,累加回调的返回值; 它是上一次调用callback时返回的累积值

　　　-- 提供初始值通常更安全

reduceRight() -- 从右到左

　　　示例： 

            [2, 5, 8, 1, 4].reduce((a,b) => a+b) // 20 == 2 + 5 +8 + 1 + 4；

            [2, 5, 8, 1, 4].reduce((a,b) => a-b) // -16 == 2 - 5 - 8 - 1 -4；

            [2, 5, 8, 1, 4].reduce((a,b) => a*b) // 320 == 2*5*8*1*4；

更多实用案例：

计算数组中元素出现的次数

            arr1.reduce(function(accumulator,curr){
                if(curr in accumulator)
                    accumulator[curr]++;
                else
                    accumulator[curr] = 1;
                return accumulator;
            },{})

String对象：

作用：用于处理文本（字符串）。

创建：

        var str = "abc"; // string   ["a", "b", "c"]
        var str = new String("abc"); // object

常用API：

属性：

length -- 字符串长度

方法：

字符：

　　　charAt(index)

　　　　　-- 返回指定索引处的字符。index默认为0。

　　　charCodeAt(index)

　　　　　-- 返回指定索引处字符的 UTF-16 编码值。index默认为0。

　　　String.fromCharCode(num1, ..., numN)

　　　　　-- 将一系列编码转换为字符串

连接：

　　　concat()

　　　　　-- 类似于数组中相同API

查找：

　　　indexOf()

　　　　　-- 类似于数组中相同API

　　　lastIndexOf()

　　　　　-- 类似于数组中相同API

截取子字符串：

　　　slice(start, end)

　　　　　-- 类似于数组中相同API

　　　substring(start, end)

　　　　　注意：

　　　　　(1)当参数为正值时，两个方法几乎一样；

　　　　　(2)substring是以两个参数中较小一个作为起始位置，较大的参数作为结束位置。因此 str.substring(1, 0) == str.substring(0, 1)；

　　　　　(3)当参数为负值时，它们就有区别了；slice会将字符串的长度与对应的负数相加，结果作为参数；substring则干脆将负参数都直接转换为0。

　　　substr(start, length)

　　　　　-- 这个方法提取指定长度的字符串。了解，不建议使用......
