对象：本地对象、内置对象和宿主对象

内置对象：

Array：数组

作用：Array 对象用于在单个的变量中存储多个值。

概念：

下标 -- 编号，从 0 开始，到 (长度-1) 结束，索引

元素 -- 数组中存放数据的空间

长度 -- 数组中存放数据的个数

创建数组对象：

    a. var array = [];
    b. var array = new Array();
    c. var array = [value1, value2, value3, ...]; // 创建同时初始元素
    d. var array = new Array(value1, value2, value3, ...);
    e. var array = new Array(size); // size 数字类型，表示数组长度

数组元素引用：

数组名[下标]

遍历数组元素：

a. 普通循环

    for (var i = 0; i < array.length; i++) {
        // array[i]
    }

b. for - in

    for (var i in array) { // i 表示数组下标的字符串内容，比如下标为 0，那i就是"0"， i 表示数组下标的字符串内容，比如下标为 0，那i就是"0"
        // array[i]
    }

遍历对象属性：

    for (var attrName in obj) { // attrName 表示对象属性名字符串内容
        var value = obj[attrName]; // obj[attrName]获取对象属性值
    }

数组API：

属性：

length -- 获取/设置数组长度

方法：

A.添加：

　　　(1)尾部：push()

　　　　　将一个或多个元素添加到数组的末尾，并返回新数组的长度。

　　　　　// 该方法和 call() 或 apply() 一起使用时，可应用在类似数组的对象上。

　　　(2)头部：unshift()

　　　　　将一个或多个元素添加到数组的开头，并返回新数组的长度。

　　　　　// 用法类似于push()

　　　(3)指定索引处：

            arrayObject.splice(index, 0, item1,.....,itemX)

B.删除：

　　　(1)尾部：pop()

　　　　　从数组中删除最后一个元素，并返回该元素的值。此方法更改数组的长度。

　　　(2)头部：shift()

　　　　　从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度。

　　　(3)指定索引处：arrayObject.splice(index, howmany, item1,.....,itemX)

　　　　　(3.1)注意：添加/删除元素都会动态改变数组长度及数组下标

　　　　　(3.2)splice(index,howmany,item1,......,itemX)

　　　　　(3.3)index 是要操作的索引位置，会影响之后数组元素的索引

　　　　　(3.4)howmany 为0时 添加元素，为大于0时 删除元素。

C.排序：

　　　(1)reverse() -- 反转

　　　　　将数组中元素的位置颠倒。

　　　(2)sort(sortBy) -- 排序(升序，默认字符串Unicode编码顺序)

　　　　　sortBy 可选排序函数，该函数要比较两个值，然后返回一个用于说明这两个值的相对顺序的数字。比较函数应该具有两个参数 a 和 b，其返回值如下：

　　　　　　　(2.1)若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前，则返回一个小于 0 的值。

　　　　　　　(2.2)若 a 等于 b，则返回 0。

　　　　　　　(2.3)若 a 大于 b，则返回一个大于 0 的值。

                    function sortNumber(a,b)
                    {
                        return a - b; // 升序排列数字数组
                    }

D.连接：

　　　concat() -- 数组连接，返回值为新数组

　　　　　var arrNew = arr1.concat(arr2);

　　　join() -- 连接数组元素，返回字符串，默认使用 ,(逗号) 连接数组元素

E.截取：

　　　slice(start, end)

　　　　　-- 截取 start 开始到 end 结束之间的元素，可省略 end 表示截取到数组末尾。

　　　　　-- 包括开始索引处元素，但不包括结束索引处元素。

　　　　　-- start，end 也可以取负值

　　　　　-- 返回新数组，不影响原数组

F.转换：

　　　toString()

　　　　　-- 返回一个字符串，表示指定的数组及其元素。

　　　　　-- 该字符串由数组中的每个元素的 toString() 返回值经调用 join() 方法连接（由逗号隔开）组成。

需要记住这些API的：1. 方法名　2. 方法作用　3. 参数　4. 返回值

排序算法：

冒泡排序：

A.相邻两个元素比较，按升序（降序）排序，如果相邻两个元素不满足排序规则，则交换顺序。

B.会经过多轮的排序，每轮会经过多次排序，通常规则是：轮数 + 当前轮比较次数 == 元素个数

C.代码：

　　　(1)二重循环

　　　(2)外层控制比较的轮数，内层控制每轮比较的次数

　　　(3)外层循环 N-1，内层循环 N-1-i (N表示数组长度，i表示外层循环变量，外/内层循环变量都从0开始)

        for(let i = 0; i < len-1; i++){
            for(let j = 0; j < len-1-i; j++){
                if(arr[j] < arr[j+1]){
                    var mid = arr[j+1];
                    arr[j+1] = arr[j];
                    arr[j] = mid;
                }
            }
        }
