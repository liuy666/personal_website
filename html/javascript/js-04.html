函数：函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。

https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide/Functions

本质：代码块

函数定义：

a. 函数声明：

    function 函数名(可选参数列表) {
        // 代码块
    }

b. 函数表达式：

    var 变量名 = function (可选参数列表) {
        // 代码块
    }
    (var 变量名 = 匿名函数;)

c. (了解) new 构造函数：

    var 变量名 = new Function("可选参数列表", "函数主体");

函数调用：

a. 直接调用

　　　函数名(可选参数列表);

b. 事件驱动

　　　xxx.onclick = 函数名;

函数调用时会先转向函数体执行代码块，当函数体执行完毕后回到调用处继续向后执行。

参数：

a. 形式参数（形参） -- 函数定义时的参数列表

b. 实际参数（实参） -- 函数调用时的参数列表

返回值：

将函数处理结果返回。返回到函数调用处，以便函数调用结束后能够继续使用到在函数体中处理的结果。

return 表达式;
