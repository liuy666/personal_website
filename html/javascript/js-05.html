函数：

a.参数：

　　　(1).形参个数与实参个数可以不相同。

　　　(2).当实参个数少于形参个数时，未传递参数在函数主体中的取值为 undefined；

　　　(3).当实参个数多于形参个数时，多余传递的参数在函数主体中暂时忽略。

b.arguments（对象）：

　　　用于函数体内，作用：表示获取到的实际参数列表信息。

c.如果需要获取实参列表中某个具体参数值时：

　　　(1).可以使用arguments[index]方式来获取具体参数值，index 相当于是参数的编号，该编号从0开始。

　　　(2).可以使用 arguments.length 来获取实参个数。

　　　(3).可以使用 函数名.length 来获取形参个数（了解）

d.返回值：return 表达式;

　　　作用：

　　　　　(1).将表达式的值返回到函数调用处

　　　　　(2).结束函数的执行

　　　return;

　　　　　(1).常用于结束函数执行；函数返回值为 undefined。

　　　　　(2).如果函数未显式使用 return 返回，则函数的返回值为 undefined。

f.跳转语句：

　　　break;

　　　　　可用于 switch 与循环结构，表示退出整个语法结构

　　　continue;

　　　　　只能用于循环结构，表示结束当前次循环

　　　return 表达式;

　　　　　用于函数中，表示结束函数执行，同时将表达式结果返回到函数调用处

作用域：

a.全局作用域：

定义在函数体外部的变量，可以整个脚本程序全局范围内使用的变量。

b.局部作用域(函数级作用域)：

函数体内部所定义的变量，其作用范围只在函数体的 {} 范围内有用，超出函数 {} 后，则不能再使用。

局部作用域中的变量可以和全局变量同名，在函数体中优先使用局部变量。

c.块级作用域：

IIFE： Immediately Invoked Function Expression 立即调用函数表达式（匿名自执行函数、自执行匿名函数）

使用IIFE实现块级作用域.

ES6: let 变量名 = 变量值; const 变量名 = 变量值; // 定义的是常量

d.作用域链：

关于变量查找的问题：要使用某个变量，首先在自己当前作用域下查找是否存在，如果存在则直接使用；

如果不存在，则会向上层作用域中查找；

如果上层作用域中存在，则直接使用，如果不存在，则继续向上层的上层作用域查找（作用链）。

如果作用域链中不存在使用的变量，则提示错误。

e.声明提升(预解析)：

将 var 定义变量的声明部分向其作用域的最顶部提升，注意提升的是声明，赋值仍然保留在原来位置上。

对于函数也存在声明提升，会将函数声明方式定义出来的函数整体(包括函数声明与函数主体)向作用域的最顶部提升。

对于函数表达式方式定义出来的函数，仅提升变量声明部分，不提升赋值部分。
