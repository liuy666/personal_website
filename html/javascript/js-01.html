3．编写JS及如何运行JS

  • 类比于 CSS的编写及运行
  • 内联js
  • 内部js:

    <script type="text/javascript"></script>

　　　注意：type 属性可以省略，默认 type 取值为 "text/javascript", script 元素可以写在文档中任意位置，建议写在 body 标签前。

  • 引入外部js文件：

    <script src="外部JS资源路径"></script>

　　　注意：script 必须独立书写,开始与结束标签间不添加其它JS代码。

  • 运行：

　　　JS引擎 -- chrome -- V8

  • 输出:

    console.log() -- 控制台打印输出
    console.dir() -- 详细打印
    console.table() -- 表格形式打印
    confirm() -- 显示一个确认对话框
    alert() -- 显示一个警告对话框
  • 输入：

　　　document.write() --将一个文本字符串写入文档流

　　　注意：向文档流输入，可能会打乱页面原有布局

　　　prompt() -显示一个对话框,用来提示用户输入文字

　　　注意：点击"取消"或者关闭返回 NULL

　　　document.getElementById("xxx") 文本框输入【DOM操作】

　　　innerText / innerHTML

  • 注释：

　　　单行： //

　　　多行： /* */

  • js文件内常用注释格式：

    /**
     * @param {type} parameter
     * @return {type}
     */

4． 变量(variate)的概念及基本运算

变量：

　　　内存中数据存储的空间，该空间中的数据可以随着运算而发生改变

变量三要素：

　　　变量名 -- 别名

　　　变量值 -- 数据

　　　变量类型 -- 区分不同类型的数据，空间大小

变量定义：

　　　语法：var 变量名 = 变量值;

变量申明：

　　　var 变量名;

变量赋值：

　　　变量名 = 变量值;

变量类型：

　　　JS 是弱类型（动态类型）的语言，变量的类型是运行时动态确定的。

　　　在变量定义时不需要显式指明变量的数据类型。

分类：

　　　基本数据类型（原始数据类型）：

        String
        Boolean -- false / true
        Number
        Undefined
        Null
        Symbol

　　　引用数据类型：Object

求解表达式的数据类型:

　　　typeof 表达式

　　　注意：typeof 相当于是个运算符

typeof 所求解的结果的数据类型为字符串，它有一下几种取值：

        String --- "string"
        Boolean --- "boolean"
        Number --- "number"
        Undefined -- "undefined"
        Null -- "object"
        Symbol -- "symbol"
        Object --"object"
        Function(){} -- "function"  注意：函数也可以求解数据类型

5． 变量命名规则和关键字的介绍

关键字：

　　　语法中已有定义的有特殊意义的单词

变量命名规则：

　　　a. 可取 字母、数字、下划线_、$ 这些字符

　　　b. 不能以数字开头

　　　c. 不能是关键字

变量命名规范：

　　　a. 取简洁的英文单词命名

　　　b. 第一个单词首字符小写，其它单词首字符大写，除首字符以外其它字符全部小写

6． 变量不同类型之间的自动、手动类型转换

数据类型转换：

　　　自动(隐式)类型转换

　　　手动(显式、强制)类型转换:

　　　　　parseInt()

　　　　　parseFloat()

　　　　　-- 通常用于将字符串解析为数字

　　　　　-- 有一定的容错能力，如果第一个字符为非兼容格式的数字字符，则返回 NaN

　　　　　-- NaN 是 Number 类型的一个特殊取值(not a number)

　　　　　isNaN(xx)

　　　　　-- 判断参数是否为非数字结果

　　　　　-- 返回 true 表示 非数字（不是数字）

　　　　　-- 返回 false 表示 数字

　　　　　Number()

　　　　　-- 将其它类型转换为数字

　　　　　a. String

　　　　　　　完全兼容的数字格式 --> 对应的数字

　　　　　　　不兼容的格式 --> NaN

　　　　　　　" "(空字符串) --> 0

　　　　　b. Boolean

　　　　　　　true --> 1

　　　　　　　false --> 0

　　　　　c. Undefined

　　　　　　　undefined --> NaN

　　　　　d. Null

　　　　　　　null --> 0

　　　　　e. Object

　　　　　　　{} --> NaN

　　　　　f. Symbol

　　　　　　　symbol --> NaN

　　　　　g. Function

　　　　　　　function(){} --> NaN

7． 赋值、关系、算术等运算符的讲解

赋值运算符：=

　　　将右边表达式的值赋给左边的变量保存

算术运算符(二目运算符) ：

　　　+ - * / %(模)

　　　注意：字符串的+表示字符串拼接

自增、自减　++ --：

　　　前缀(++i)：先运算后引用

　　　后缀(i++)：先引用后运算

　　　示例：

        var i = 5;
        var j = i++ + ++i + ++i + i++ + i + i++;
        i = ?   j = ?

关系(比较)：

　　　> < >= <= == === != !==

　　　==：

　　　　　在数据类型一致的情况下，比较值是否相同；

　　　　　数据类型不一致的情况下，先尝试转换为同种数据类型，再比较值是否相同

　　　===：

　　　　　在数据类型一致的情况下，比较值是否相同；

　　　　　数据类型不一致的情况下，直接返回 false

逻辑运算符：

　　　&&(并且) ||(或者) !(非，单目运算符)

短路运算符：

　　　&&：第一个表达式为 false 则不再进行第二个表达式运算

　　　||：第一个表达式为 true 则不再进行第二个表达式运算

　　　示例：求某年是否是闰年 <闰年：能被4整除但不能被100整除 或 能被400整除>

三目运算符：

　　　条件 ? true : false

　　　示例：求某年2月有多少天

复合运算符：

　　　+= -= *= /= %=

其它单目运算符(了解)：

　　　+ - ~ & | >> () .

　　　注意：这里的+ - 不同于上面的双目运算符

　　　示例：+New Date() -(function(){})()
