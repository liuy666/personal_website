数字 -->

  • 0、NaN --> false
  • 其它 --> true

字符串 -->

  • "" --> false
  • 非空字符串 --> true

流程控制语句：

  • 顺序结构
  • 选择结构（条件结构）
  • 循环结构

选择结构：

1.if

语法：

    if (条件表达式) {
        // 条件表达式为 true 时执行的语句块
    }

2.if - else

语法：

    if (条件表达式) {
        // 条件表达式为 true 时执行的语句块
    } else {
        // 条件表达式为 false 时执行的语句块
    }

3.多重 if : if - else if - else

语法：

    if (条件表达式1) {
        // 条件表达式1为 true 时执行的语句块
    } else if (条件表达式2) {
        // 条件表达式2为 true 时执行的语句块
    } else if (条件表达式3) {
        // 条件表达式3为 true 时执行的语句块
    } else {
        // 前述条件均不满足时执行的语句块
    }

注意：当 if 或 else 语句块中只有一条可执行语句时，也可以省略 {} 的书写

4.switch结构：

语法：

    switch (表达式) {
        case 常量1:
            // 语句块1
            break;
        case 常量2:
            // 语句块2
            break;
        .....
        default:
            // 缺省块
            break;
    }

switch 结构实现的是等值(===)条件的判断；

case 语句块中的 break; 语句是可以省略的，当省略 break; 语句时，可能导致 case 语句贯穿：

        switch (表达式) {
            case 常量1:
            case 常量2:
                // 语句块
                break;
            ......
            default:
                // 缺省块
                break;
        }
