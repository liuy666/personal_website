BOM：Browser Object Model

浏览器对象模型，使 JavaScript 有能力与浏览器“对话”

Window 对象：

代表浏览器中打开的窗口，使用 window 来引用 Window 对象。当调用 window 对象的属性与方法时，可以省略 (window.) 的书写

Location 对象

它是window对象的一部分:表示URL（网址）

使用 window.location 或 location 来获取

URL格式：

协议://域名:端口/资源路径?查询字符串#hash

HTTP: 默认端口 80

HTTPS: 默认端口 443

查询字符串格式：

key1=value1&key2=value2&key3=value3

属性：

hash 设置或返回从井号 (#) 开始的 URL（锚）。 #hash

host 设置或返回主机名和当前 URL 的端口号。 域名:端口

hostname 设置或返回当前 URL 的主机名。 域名

href 设置或返回完整的 URL。 协议://域名:端口/资源路径?查询字符串#hash

pathname 设置或返回当前 URL 的路径部分。 /资源路径

port 设置或返回当前 URL 的端口号。 端口

protocol 设置或返回当前 URL 的协议。 协议

search 设置或返回从问号 (?) 开始的 URL（查询部分）。 ?查询字符串

API：

reload() 重新加载，刷新

assign() 加载新的文档。会添加历史访问记录

replace() 用新的文档替换当前文档。不会添加历史记录

History 对象：表示访问历史

使用 window.history 或 history 来获取。

API:

back() 后退

forward() 前进

go(number) +number前进 -number后退

Navigator 对象： 包含有关浏览器的信息。navigator

Document 对象：代表浏览器中打开的文档

使用 window.document 或 document 来获取

window.innerWidth 获取视口宽度（可视的文档窗口尺寸，包括滚动条）

window.innerHeight 获取视口高度

API：

alert()

prompt()

confirm() -- 确认框

open() --弹出窗口打开新页面

close()

计时器：

var id = setTimeout(fn, duration)

　　　一次计时，到达计时时间后执行一次计时函数

　　　fn -- 计时时间到达时执行的函数

　　　duration -- 计时时间

var id = setInterval(fn, duration)

周期计时，每隔计时时间都会执行计时函数

清除计时器：

clearTimeout(id)

clearInterval(id)

注意：setTimeout()、setInterval()都是异步执行计时任务，JS是单线程的程序。

DOM：Document Object Model 文档对象模型

https://developer.mozilla.org/zh-CN/docs/Glossary/DOM

在 JavaScript 中可以操纵文档内容：添加、修改、删除、克隆、查询节点......

document 指代整个文档。

查找元素节点：

document.getElementById() 
    -- 通过元素的 id 属性来查找元素，返回查找到的DOM对象，如果不存在对应id的元素，则返回null

document.getElementsByTagName() 
    -- 通过元素名查找元素，返回 HTMLCollection 对象，该对象是类(似于)数组对象，对象中保存了查找到的DOM对象

document.getElementsByClassName() 
    -- 通过类名查找元素(ES5新增)，返回 HTMLCollection 对象

document.getElementsByName() 
    -- 根据元素 name 属性来查找元素(了解)

element.getElementsByTagName() 
    -- 在element元素后代中通过元素名查找元素，返回 HTMLCollection 对象，该对象是类(似于)数组对象，对象中保存了查找到的DOM对象

element.getElementsByClassName() 
    -- 在element元素后代中通过类名查找元素(ES5新增)，返回 HTMLCollection 对象

扩展：

document.querySelector(selector)
    -- 返回文档中匹配指定的选择器组的第一个元素

document.querySelectorAll(selector)
    -- 返回与指定的选择器组匹配的文档中的元素列表
