循环

循环结构：重复

循环特征：

　　- 循环条件

　　- 循环操作

循环使用步骤：

　　- 找出循环条件与循环操作

　　- 代入循环的语法结构

　　- 判断是否能够退出循环

while 循环流程图：

语法：

while (条件表达式) {
    // 条件表达式为 true 时执行的语句块
    // 循环操作：循环体
}

do-while 循环流程图：

语法：

do {
    // 循环体：循环操作
} while (条件表达式);

while 与 do-while 区别：

执行流程：

　　while : 先判断后执行

　　do-while : 先执行后判断

for 循环流程图：

语法：

for (变量初始化; 条件表达式; 变量更新) {
    // 循环体
}

三个表达式均可以省略：

  1. 省略－－变量初始化 ，则通常在 for 循环结构前实现 变量初始化；
  2. 省略－－条件表达式时，for 循环成为死循环，通常需要在循环主体中构建退出循环的条件然后退出
  3. 省略－－变量更新，则通常在 循环主体中实现变量更新

适合场合：

  1. for 常用于循环次数确定的情况
  2. while、do-while 常用于循环次数不确定的情况，while 用于先判断后执行的场景，do-while用于先执行后判断的场景

跳转语句：

  • break;　退出整个循环结构
  • continue;　结束当前次循环，还会继续判断条件执行下一次循环
