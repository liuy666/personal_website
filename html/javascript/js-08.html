Date对象：用于处理日期和时间

创建对象：

            var date = new Date(); -- 系统当前时间

            var date = new Date(milliseconds);
            milliseconds 是距离 1970-1-1 0:0:0(标准时) 以来的毫秒值

            var date = new Date("yyyy-MM-ddTHH:mm:ss");
            注意：T是固定的格式，对于月、日、时、分、秒不够2位必须补0，如"2015-02-06T01:01:01"

            var date = new Date(year, month[, day[, hour[, minutes[, seconds[, milliseconds]]]]]);
            注意：这里的参数可以是1位数，new Date(1995, 1, 1, 1, 1, 1);

常用API：

A.获取：

            getFullYear() 四位年份
            getMonth() 月，返回 0~11
            getDate() 日
            getDay() 星期几，返回 0~6 --容易混淆
            getHours() 时
            getMinutes() 分
            getSeconds() 秒
            getMilliseconds() 毫秒

getTime() -- 获取1970-1-1以来总的毫秒值(时间戳timestamp),表示从1970年1月1日0时0分0秒（UTC，即协调世界时）距离该日期对象所代表时间的毫秒数。

如果日期对象创建后不做更改，则这个值固定，这也是区别于 Date.now()的地方。

B.设置：

            setFullYear() 四位年份
            setMonth() 月
            setDate() 日
            setHours() 时
            setMinutes() 分
            setSeconds() 秒
            setMilliseconds() 毫秒

setTime(timeValue) -- 把一个日期时间赋值给另一个 Date 对象,timeValue 是一个毫秒值

C.转换：

            Date.parse() -- 将字符串所表示的日期时间转换成毫秒值
            toString() -- 将日期时间对象转换为字符串格式，使用GMT时区
            toUTCString() -- 将日期时间对象转换为字符串格式，使用UTC时区

与 1970-1-1 0:0:0 相关的毫秒值：

            dateObject.getTime()

            dateObject.valueOf() -- 该方法的功能和 Date.prototype.getTime() 方法一样。该方法通常在 JavaScript 内部被调用，而不是在代码中显式调用。

            +new Date() --> +date // var date = new Date()

            Date.parse()

ES5新增：

Date.now() -- 返回自1970年1月1日 00:00:00 UTC到【当前时间】的毫秒数。【看概念似乎和getTime()很相似？】

注意： 

            Date.now() === +new Date()
            Date.now() !== dateObject.getTime()

Math 对象：用于执行数学任务,Math对象没有构造函数，即不能使用 new Math() 来创建Math对象。

Math相当于是 var Math = {}

常用API：

                Math.random() // 产生随机数
                Math.floor() // 向下取整
                Math.ceil() // 向上取整
                Math.round() // 四舍五入
                Math.pow(base,exponent) // 指数幂
                Math.sqrt() // 平方根
                Math.abs() // 绝对值 (absolute value)
                Math.min()/Math.max() // 求一组数的最大/最小值

JSON 对象 -- ES5新增：

JSON

JavaScript Object Notation

JavaScript对象表示法

JSON 是存储和交换文本信息的语法。类似 XML。

JSON中存储的是文本信息,它是一种字符串。

JSON.parse() -- 将JSON字符串内容解析为JS值

JSON.stringify() -- 将JS值转换为对应格式的JSON字符串

注意：如果在 JSON 字符串中要表示对象，则对象的属性名必须使用【双引号】包含,如： '{"age" : 18, "name" : "liu"}'

JSON字符串中要表示JS的字符串值，也必须使用【双引号】包含,如： '"abcdef"'

内置对象：Array String Date Math JSON Number
